# 👨‍💻 Portafolio Personal - Santiago De Rosa (SERV-DRV)

¡Bienvenido a mi portafolio personal! Este proyecto fue desarrollado desde cero utilizando **React Native y Expo** para crear una experiencia moderna, rápida y con un diseño de alto contraste (Blanco y Negro con acentos azules) orientado a resultados.

## 🎯 Objetivo
El propósito de este portafolio es centralizar mis capacidades como desarrollador **Full Stack**, mostrando de manera interactiva mi información profesional, mis habilidades técnicas y los proyectos reales que he construido.

## 🛠️ Tecnologías Principales (Mi Stack)
- **Frontend:** React, React Native, HTML5, CSS3, JavaScript.
- **Backend:** Java, C#, Node.js.
- **Bases de Datos:** MongoDB, MySQL, PostgreSQL.
- **Control de Versiones:** Git & GitHub.

## 📁 Estructura del Proyecto
- Arquitectura modular y escalable.
- Componentes reutilizables.
- Animaciones fluidas utilizando `react-native-reanimated`.
- Renderizado de componentes dinámico.

## 🚀 Proyectos Destacados Incluidos
1. **Agenda-Web:** Sistema web de gestión de contactos y tareas.
2. **WorkAs:** Sistema integral para gestión de contratos y facturación.
3. **Gestor Restaurante:** Plataforma web y móvil para administración de restaurantes.
4. **Deporte Mania:** E-commerce de artículos deportivos.
5. **Kinal-Familiar:** Sistema para control de economía y presupuesto.
